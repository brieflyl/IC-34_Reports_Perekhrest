Висновки

На цій лабораторній роботі я навчилась працювати з HTML-документами, таблицями, зображеннями,
посиланнями, списками, формами. Створила три репозиторії: для лабораторних, для самостійних робіт
та для звітних документів.

Контрольні питання

1. Яка структура HTML-документу
Зазвичай HTML-документ складається з основних тегів html, head, title, body

2. Дати визначення тега
Тег - це елемент HTML, який визначає вміст і структуру веб-сторінки

3. Які теги використовуються при роботі з таблицями
table - для оголошення таблиці, tr - для оголошення рядка таблиці, th - для оголошення заголовка стовпця або рядка таблиці, td - для оголошення комірки таблиці
tr - для оголошення рядка таблиці

4. Які теги використовуються при роботі з зображеннями
img - для відображення зображення на сторінці

5. Які теги використовуються при роботі зі списками
ul - для оголошення невпорядкованого списку, ol - для оголошення впорядкованого списку, li - для оголошення елемента списку

6. Перерахувати засоби опису таблиць в html
Раніше було описано теги, які використовуються для опису таблиць, до них можна додати також
caption - для опису таблиці, table border - встановлює ширину межі таблиці, table cosplan - об'єднує декілька комірок по горизонталі,
table rowspan - об'єднує декілька комірок по вертикалі
